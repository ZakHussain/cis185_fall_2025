The Document Object Model (DOM)

The DOM is a tree-like representation of your HTML document that JavaScript can interact with.

HTML Code:

<html> <body> <h1 id="title">Hello</h1> <p class="text">Welcome</p> </body> </html>

DOM Tree:

Key Concepts:

  • Parent-Child Relationship: body is the parent of h1 and p
  • Siblings: h1 and p are siblings (same parent)
  • Traversal: JavaScript can navigate up, down, and across the tree

Think of the DOM as a family tree where HTML elements are related as parents, children, and siblings

4 / 15

More Complex DOM Structure

Real websites have nested elements creating deeper DOM trees:

HTML Code:

<body> <div id="header"> <h1>My Site</h1> <nav> <a href="#">Home</a> <a href="#">About</a> </nav> </div> <div class="content"> <p>Welcome!</p> <button id="btn">Click</button> </div> </body>

DOM Tree Visualization:

Important DOM Relationships:

  • Parent of nav: div#header
  • Children of body: div#header and div.content
  • Siblings: The two anchor (a) tags are siblings
  • Descendants: All elements inside body are its descendants

JavaScript can navigate this tree: From any element, you can access its parent, children, or siblings!

Adding JavaScript to HTML

Method 1: Inline JavaScript

<button onclick="alert('Hello!')">Click Me</button>

Method 2: Internal JavaScript

<script> function sayHello() { alert('Hello from internal script!'); } </script>

Method 3: External JavaScript (Recommended)

<script src="script.js"></script>

Best Practice: Place <script> tags at the end of <body> for better page loading

Selecting Elements

Before we can manipulate elements, we need to select them from the DOM.

Main Selection Methods:

1. getElementById()

// HTML: <h1 id="title">Hello</h1> let heading = document.getElementById('title');

2. getElementsByClassName()

// HTML: <p class="text">Paragraph</p> let paragraphs = document.getElementsByClassName('text');

3. querySelector()

// Selects first matching element let firstPara = document.querySelector('.text'); let heading = document.querySelector('#title');
8 / 15

More Selection Methods

4. querySelectorAll()

// Selects ALL matching elements let allParas = document.querySelectorAll('.text'); let allDivs = document.querySelectorAll('div');

5. getElementsByTagName()

// Selects all elements with specified tag let allImages = document.getElementsByTagName('img');
Method Returns Example
getElementById Single element getElementById('header')
querySelector First match querySelector('.btn')
querySelectorAll All matches querySelectorAll('.btn')

Modifying Content

Changing Text Content

// Select the element let heading = document.getElementById('title'); // Change its text heading.textContent = 'New Title'; // Or use innerHTML for HTML content heading.innerHTML = '<em>New Title</em>';

Changing Attributes

// Change an image source let img = document.querySelector('img'); img.src = 'new-image.jpg'; img.alt = 'New description'; // Or use setAttribute img.setAttribute('src', 'new-image.jpg');

textContent vs innerHTML:
textContent = plain text only
innerHTML = can include HTML tags

Changing CSS with JavaScript

Event Handling

Events are actions that happen in the browser (clicks, key presses, mouse movements)

Common Events:

  • click - User clicks an element
  • mouseover - Mouse enters an element
  • mouseout - Mouse leaves an element
  • keydown - User presses a key
  • submit - Form is submitted
  • load - Page finishes loading

Adding Event Listeners

let button = document.getElementById('myButton'); button.addEventListener('click', function() { alert('Button was clicked!'); });
11 / 16

Event Handling Examples

Multiple Ways to Handle Events

Method 1: Inline (Not Recommended)

<button onclick="handleClick()">Click</button>

Method 2: Property Assignment

button.onclick = function() { console.log('Clicked!'); };

Method 3: addEventListener (Best Practice)

button.addEventListener('click', function() { console.log('Clicked!'); }); // Can add multiple listeners to same event button.addEventListener('click', function() { console.log('Another action!'); });

JavaScript Code (script.js)

// Select elements let heading = document.getElementById('heading'); let changeBtn = document.getElementById('changeBtn'); let colorBtn = document.getElementById('colorBtn'); // Add event listener for text change changeBtn.addEventListener('click', function() { heading.textContent = 'Text Changed!'; }); // Add event listener for color change colorBtn.addEventListener('click', function() { heading.style.color = 'blue'; heading.style.fontSize = '40px'; }); // Add mouseover effect heading.addEventListener('mouseover', function() { heading.style.backgroundColor = 'yellow'; }); heading.addEventListener('mouseout', function() { heading.style.backgroundColor = 'white'; });
