JavaScript Arrays

Creating and Manipulating Arrays

Arrays are ordered collections that can hold multiple values:

// Creating arrays let fruits = ["apple", "banana", "orange"]; let numbers = [1, 2, 3, 4, 5]; // Common methods fruits.push("grape"); // Add to end fruits.pop(); // Remove from end fruits.unshift("mango"); // Add to beginning fruits.shift(); // Remove from beginning

Key Point: Arrays are zero-indexed. The first element is at index 0.

Advanced Array Methods

forEach, map, filter, reduce

let numbers = [1, 2, 3, 4, 5]; // forEach - iterate over each element numbers.forEach(num => console.log(num)); // map - transform each element let doubled = numbers.map(num => num * 2); // Result: [2, 4, 6, 8, 10] // filter - select elements that match condition let evens = numbers.filter(num => num % 2 === 0); // Result: [2, 4] // reduce - combine all elements into single value let sum = numbers.reduce((total, num) => total + num, 0); // Result: 15

JavaScript Objects

Creating and Accessing Object Properties

Objects store data as key-value pairs:

// Creating an object let student = { name: "Alice", age: 20, major: "Computer Science", gpa: 3.8 }; // Accessing properties console.log(student.name); // Dot notation console.log(student["age"]); // Bracket notation // Adding new properties student.email = "alice@example.com"; // Deleting properties delete student.gpa;

Object Methods

Functions Inside Objects

let car = { brand: "Toyota", model: "Camry", year: 2024, // Method using 'this' keyword getInfo: function() { return `${this.year} ${this.brand} ${this.model}`; }, // Arrow function method (ES6) startEngine: () => { console.log("Engine started!"); } }; console.log(car.getInfo()); // "2024 Toyota Camry"

Remember: The this keyword refers to the object itself.

The Date Object

Working with Dates and Times

// Creating Date objects let now = new Date(); // Current date/time let specific = new Date("2024-10-31"); // Specific date // Common Date methods now.getFullYear(); // 2024 now.getMonth(); // 0-11 (0=January) now.getDate(); // 1-31 now.getDay(); // 0-6 (0=Sunday) now.getHours(); // 0-23 now.getMinutes(); // 0-59 // Formatting dates now.toDateString(); // "Tue Oct 28 2025" now.toLocaleDateString(); // "10/28/2025"

The Math Object

Mathematical Operations and Constants

// Math constants Math.PI; // 3.141592653589793 Math.E; // 2.718281828459045 // Rounding methods Math.round(4.7); // 5 Math.ceil(4.1); // 5 (round up) Math.floor(4.9); // 4 (round down) // Common operations Math.abs(-5); // 5 (absolute value) Math.pow(2, 3); // 8 (2 to the power of 3) Math.sqrt(16); // 4 (square root) Math.random(); // Random number 0-1 // Random integer between min and max function randomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; }

String Object Methods

Manipulating Text

let text = "Hello World"; // Case conversion text.toUpperCase(); // "HELLO WORLD" text.toLowerCase(); // "hello world" // Searching text.indexOf("World"); // 6 text.includes("Hello"); // true text.startsWith("Hello"); // true // Extracting text.slice(0, 5); // "Hello" text.substring(6, 11); // "World" // Modifying text.replace("World", "JS"); // "Hello JS" text.trim(); // Remove whitespace text.split(" "); // ["Hello", "World"]

JSON - JavaScript Object Notation

Data Exchange Format

JSON is a text format for storing and transporting data:

// JavaScript object let person = { name: "John", age: 30, city: "New York" }; // Convert object to JSON string let jsonString = JSON.stringify(person); // Result: '{"name":"John","age":30,"city":"New York"}' // Convert JSON string back to object let parsedObject = JSON.parse(jsonString); console.log(parsedObject.name); // "John"

Use Case: JSON is commonly used for APIs and data storage.

Putting It All Together

Complete Example

// Create a student management system let students = [ { name: "Alice", grades: [85, 90, 92] }, { name: "Bob", grades: [78, 82, 88] }, { name: "Carol", grades: [95, 93, 96] } ]; // Calculate average grade for each student let results = students.map(student => { let avg = student.grades.reduce((sum, grade) => sum + grade, 0) / student.grades.length; return { name: student.name, average: Math.round(avg * 100) / 100 }; }); // Find students with average > 85 let topStudents = results.filter(s => s.average > 85); console.log(JSON.stringify(topStudents, null, 2));
